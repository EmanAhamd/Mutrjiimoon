ABOUT JOB

Our company is in search of a mid to senior level full stack Laravel Developer who has the technical ability to implement front end and server side code written in the Laravel framework. Your main responsibility will be designing and implementing applications that are written in PHP. You will work closely in a team environment. JOB RESPONSIBILITIES

As a Laravel Developer at our company, you will:

  • Liaise with fellow backend and front end developers.
  • Develop and implement web applications that use the Laravel framework.
  • Implement server side logic to process frontend inputs.
  • Identify and fix bugs that are found within code.
  • Interact with SQL databases.
  • Ensure all inputs going to a database have been parameterized.
  • Ensure the appropriate security standards have been implemented.

JOB REQUIREMENTS

Our ideal Laravel Developer will have:

  • The ability to problem-solve and critically think.
  • High level of knowledge of PHP and the Laravel framework.
  • High level of knowledge of Laravel Livewire
  • High level of knowledge of front-end development (CSS/Bootstrap/Tailwind, VanillaJS/React)
  • High level of knowledge of Laravel security best practices (working with gates, permissions, roles, etc)
  • Experience working in team environment and Git best practices
  • Experience with Laravel Cashier
  • Understanding of the MVC pattern.
  • Understanding of SQL databases.
  • Experience with Laravel Spark and/or Nova/Filament a plus.
  • Good interpersonal and excellent communication skills.
